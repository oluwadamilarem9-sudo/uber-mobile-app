# Uber Style MVP App

## Project Goal
Build a ride hailing mobile app MVP similar to Uber for learning purposes.

## Tech Stack
- React Native Expo
- TypeScript
- Firebase
- Expo Router
- Zustand
- React Query
- NativeWind
- React Native Maps

## Design
- Yellow primary color
- Light gray secondary color
- Modern minimal UI
- Smooth animations
- Mobile first

## Backend
Use Firebase ONLY:
- Authentication
- Firestore
- Realtime updates
- FCM notifications
- Firebase Storage

No Node.js backend.

## Architecture Rules
- Use TypeScript only
- Keep components reusable
- Use scalable folder structure
- Avoid large files
- Use env variables
- Never hardcode API keys

## State Management
- Zustand for global state
- React Query for async/cache state

## Firestore Collections
- users
- drivers
- rides
- rideRequests
- notifications

## Ride Status
- requested
- accepted
- arriving
- ongoing
- completed
- cancelled

## Security
Generate Firestore security rules.
